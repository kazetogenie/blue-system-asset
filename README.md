# blue-system-asset
AI generated asset for Blue System's japanese word processor machine.

## FAQ
## 何に使うの？
OASYS 30 AD401の初期メニューのクローンで使います。オリジナルのものは著作権や物理的制約で使えないので似たようなものを画像生成AIで拵えただけです。

## 完成度は？
あくまで叩き台です。

このままだと試作はいいけど使いものにならないのでちゃんとあとで作り直して実機にリスペクトしたデザインにします。

## Blue System って何のことですか？
OASYS → 反転すると AOSYS → AOをひらがなにすると青→ 英語にすると Blue なので、Blue System ってことです。まだ正式名称ではありません。
